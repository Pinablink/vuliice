Vullice - Limitações dessa versão
Não processa caracteres especiais na chave nem no valor
Não processa numeros decimais
Não processa objetos internos
Não processa array 

Processa estrutura simples
Exemplo:
{
   "vullice" : "Teste"
}

Ou

{
   "vullice" : "operacao",
   "Usuario" : "Weber Alves dos Santos"
}
